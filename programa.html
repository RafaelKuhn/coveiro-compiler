DOCUMENTAÇÃO

OPERAÇÕES ARITMÉTICAS:

Para registradores N e K:

  • N_add: adiciona 1 no registrador N
  • N_sub: subtrai 1 no registrador N
  • N_mult_K: multiplica N por K e armazena o resultado em N
  • N_div_K: divide N por K e armazena o resultado em N (descarta o resto da divisão)
OPERAÇÕES DE TESTE:

Para registradores N e K:

  • N_zero: retorna TRUE quando N for ZERO, senão, retorna FALSE
  • N_maior_K: retorna TRUE quando N for maior que K, senão, retorna FALSE
  • N_menor_K: retorna TRUE quando N for maior que K, senão, retorna FALSE
